## ②課題内容（どんな作品か）
-じゃんけんで勝つとパンケーキ１枚食べれる
-負けるとパンケーキが増える
-お腹いっぱいになる前に食べ切らないといけない
-パンケーキが増えすぎたらゲームオーバー
